Create complete university exams combining pedagogically rigorous case studies (following Kupp & Mueller methodology) with the exam-typst Quarto extension. Generates exam-ready `.qmd` files with professional cover pages, automatic point summation, and structured tasks. Works in both German and English.

## Trigger Phrases

Use this skill when you want to:
- "Create an exam for [course]"
- "Build an exam with case studies"
- "Generate a written exam for [topic]"
- "Erstelle eine Klausur für [Kurs]"
- "Generiere eine schriftliche Prüfung über [Thema]"

## Core Philosophy

A strong exam combines:
1. **Realistic case studies** with engaging narratives (Lead) and clear learning objectives (Need)
2. **Progressive difficulty** from basic recall to advanced application
3. **Professional formatting** using exam-typst for credibility
4. **Clear point distribution** for transparent grading

## Exam Structure

### Complete Exam Components

1. **YAML Frontmatter** with exam configuration
2. **Automatic Cover Page** (generated by exam-typst)
3. **2-4 Case Studies** with increasing difficulty
4. **Tasks structured as `.aufgabe` and `.teilaufgabe`**
5. **Solutions in separate document** (optional)

## Case Study Development for Exams

### Adapting Kupp & Mueller for Exams

**Key Differences from Teaching Cases:**
- **Shorter context** (1-2 paragraphs max, not 3-4)
- **Time pressure** (students have limited time)
- **Self-contained** (all needed info in the case, no exhibits)
- **Clear tasks** (no open-ended discussions)
- **Point-based** (every task has explicit points)

**Keep from Kupp & Mueller:**
- ✅ Clear protagonist
- ✅ Immediate Issue (decision point)
- ✅ Realistic scenario
- ✅ Neutral presentation (no solutions revealed)
- ✅ Past tense, third person

**Adapt for Exams:**
- ✅ Compress context (students must read quickly)
- ✅ Include all data in text (no separate exhibits)
- ✅ Closed-ended tasks (calculable/gradable)
- ✅ Progressive difficulty within case

### Exam-Optimized Case Structure

```markdown
::: {.aufgabe}
## Case Study: [Title]

### Kontext

[Opening paragraph: Protagonist + Organization + Time + Immediate Issue - MAX 150 words]

[Data paragraph: All numbers, facts, and context needed - MAX 100 words]

::: {.teilaufgabe points=X}
**Aufgabe X.1:** [Question]
:::

::: {.teilaufgabe points=X}
**Aufgabe X.2:** [Question]
:::
:::
```

## Task Design by Difficulty

### Basic Tasks (8-10 points)
- **Cognitive level:** Recall, comprehension
- **Time:** 5-8 minutes
- **Examples:**
  - Define concept and identify in case
  - Simple calculation (one formula)
  - Direct application of rule

```markdown
::: {.teilaufgabe points=8}
**Aufgabe 1.1:** Berechnen Sie die Abschreibung nach der linearen Methode für die ersten drei Jahre.

*Gegeben:* Anschaffungskosten €150.000, Nutzungsdauer 5 Jahre, Restwert €10.000
:::
```

### Intermediate Tasks (10-15 points)
- **Cognitive level:** Application, analysis
- **Time:** 10-15 minutes
- **Examples:**
  - Multi-step calculations
  - Comparison of alternatives
  - Explanation with calculation

```markdown
::: {.teilaufgabe points=12}
**Aufgabe 2.2:** Vergleichen Sie die beiden Abschreibungsmethoden hinsichtlich ihrer Auswirkung auf das Eigenkapital in Jahr 1. Begründen Sie Ihre Antwort mit Berechnungen.
:::
```

### Advanced Tasks (15-20 points)
- **Cognitive level:** Evaluation, synthesis
- **Time:** 15-20 minutes
- **Examples:**
  - Strategic recommendations with justification
  - Trade-off analysis
  - Complex multi-concept problems

```markdown
::: {.teilaufgabe points=18}
**Aufgabe 3.3:** Der CFO muss sich zwischen Methode A und B entscheiden. Empfehlen Sie eine Methode unter Berücksichtigung der geplanten Kreditaufnahme und begründen Sie Ihre Empfehlung mit mindestens drei betriebswirtschaftlichen Argumenten.
:::
```

## Point Distribution Guidelines

### Total Exam Points
- **60 minutes:** 60-80 points
- **90 minutes:** 90-120 points
- **120 minutes:** 120-150 points

**Rule of thumb:** ~1 minute per point

### Points per Case Study
- **Basic case:** 25-30 points (3 tasks)
- **Intermediate case:** 35-40 points (3-4 tasks)
- **Advanced case:** 40-50 points (4-5 tasks)

### Difficulty Distribution
For a balanced exam:
- **30-40%** Basic tasks (understanding, recall)
- **40-50%** Intermediate tasks (application, analysis)
- **20-30%** Advanced tasks (evaluation, synthesis)

## Workflow: From Teaching Material to Exam

### Step 1: Identify Topics
- Review course content (lecture .qmd files)
- Select 3-4 core concepts
- Ensure coverage of Trinity of Depth (Theory, Norms, Practice)

### Step 2: Create Case Leads
For each concept:
- Find realistic company/situation
- Define protagonist
- Establish immediate issue
- Keep context brief (exam constraint)

### Step 3: Design Task Progression
Within each case:
- Task 1: Basic (understanding)
- Task 2: Intermediate (application)
- Task 3: Advanced (evaluation)

### Step 4: Balance Exam
- Check total points (~1 min/point)
- Verify difficulty distribution (30/40/30)
- Ensure topic coverage
- Review time feasibility

### Step 5: Generate Solutions
Create separate solutions document:
```yaml
---
title: "Lösungen: [Course Name]"
subtitle: "[Semester]"
format: pdf
---
```

# Klausur: Investitionsrechnung und Finanzierung

::: {.aufgabe}
## Case Study 1: Maschinenersatz bei der TechPro GmbH

### Kontext

Maria Schneider, Produktionsleiterin der TechPro GmbH in Würzburg, stand im Februar 2025 vor einer wichtigen Entscheidung. Die bestehende Produktionsmaschine (Anschaffung 2020 für €80.000) zeigte zunehmend Verschleißerscheinungen und verursachte höhere Wartungskosten. Eine neue, effizientere Maschine würde €120.000 kosten, aber die jährlichen Betriebskosten von €15.000 auf €8.000 senken.

Die alte Maschine hatte noch einen Buchwert von €32.000 (lineare Abschreibung über 10 Jahre) und könnte für €25.000 verkauft werden. Die neue Maschine würde ebenfalls über 10 Jahre linear abgeschrieben. Der kalkulatorische Zinssatz des Unternehmens beträgt 8%. Maria muss dem Vorstand bis Ende der Woche eine fundierte Empfehlung vorlegen.

::: {.teilaufgabe points=8}
**Aufgabe 1.1:** Berechnen Sie die durchschnittliche jährliche Kapitalbindung der neuen Maschine bei linearer Abschreibung und einem Restwert von €10.000.
:::

::: {.teilaufgabe points=12}
**Aufgabe 1.2:** Ermitteln Sie die jährlichen Kostenersparnisse durch die neue Maschine unter Berücksichtigung der eingesparten Betriebskosten und der zusätzlichen Abschreibungen. Berücksichtigen Sie dabei den Buchverlust aus dem Verkauf der alten Maschine im ersten Jahr.
:::

::: {.teilaufgabe points=15}
**Aufgabe 1.3:** Führen Sie eine Kostenvergleichsrechnung durch und geben Sie eine begründete Empfehlung ab, ob die Investition aus wirtschaftlicher Sicht sinnvoll ist. Nennen Sie mindestens zwei weitere qualitative Faktoren, die Maria bei ihrer Entscheidung berücksichtigen sollte.
:::
:::

::: {.aufgabe}
## Case Study 2: Finanzierungsentscheidung der StartInnovate AG

### Kontext

Die StartInnovate AG, ein Technologie-Startup aus München, hatte im März 2025 ihre Series-A-Finanzierungsrunde erfolgreich abgeschlossen. Der CFO, Thomas Weber, musste nun entscheiden, wie die geplante Expansion in Höhe von €5 Millionen finanziert werden sollte. 

Option A: Bankkredit über €5 Mio. zu 6% p.a., Laufzeit 5 Jahre, erfordert Sicherheiten im Wert von €3 Mio.

Option B: Venture Capital Investment: €5 Mio. für 25% der Unternehmensanteile, keine Sicherheiten erforderlich, aber Mitspracherechte der Investoren bei strategischen Entscheidungen.

Das Unternehmen erwartete einen EBIT von €800.000 im ersten Jahr nach der Expansion. Die Eigenkapitalquote lag aktuell bei 35%, der Unternehmenswert wurde auf €15 Millionen geschätzt.

::: {.teilaufgabe points=10}
**Aufgabe 2.1:** Berechnen Sie für beide Optionen die Eigenkapitalquote nach der Finanzierung. (Hinweis: Bei Option B erhöht sich das Eigenkapital um €5 Mio., bei Option A das Fremdkapital.)
:::

::: {.teilaufgabe points=15}
**Aufgabe 2.2:** Ermitteln Sie die Auswirkungen beider Finanzierungsoptionen auf das Ergebnis nach Steuern im ersten Jahr. Gehen Sie von einem Steuersatz von 30% aus und berücksichtigen Sie die steuerliche Abzugsfähigkeit von Fremdkapitalzinsen.
:::

::: {.teilaufgabe points=20}
**Aufgabe 2.3:** Analysieren Sie beide Finanzierungsalternativen hinsichtlich folgender Kriterien:
a) Finanzielle Flexibilität (3 Punkte)
b) Unternehmerische Unabhängigkeit (3 Punkte)
c) Risiko für das Unternehmen (3 Punkte)

Geben Sie eine begründete Empfehlung ab, welche Option Thomas Weber wählen sollte, wenn das Unternehmen geplant in 3 Jahren an die Börse gehen möchte. (11 Punkte)
:::
:::

::: {.aufgabe}
## Case Study 3: Investitionsentscheidung unter Unsicherheit

### Kontext

Dr. Lisa Hoffmann, Geschäftsführerin der GreenEnergy Solutions GmbH in Regensburg, musste im April 2025 über eine strategische Investition in eine neue Produktionslinie für Solarmodule entscheiden. Die Investition von €2,5 Millionen würde zusätzliche Erträge generieren, deren Höhe jedoch stark von der zukünftigen Energiepolitik abhing.

Szenario "Optimistisch" (Wahrscheinlichkeit 30%): Jährliche Rückflüsse von €800.000 für 5 Jahre
Szenario "Realistisch" (Wahrscheinlichkeit 50%): Jährliche Rückflüsse von €550.000 für 5 Jahre
Szenario "Pessimistisch" (Wahrscheinlichkeit 20%): Jährliche Rückflüsse von €350.000 für 5 Jahre

Der kalkulatorische Zinssatz beträgt 10%. Lisa muss dem Aufsichtsrat eine Entscheidung präsentieren, die sowohl die Chancen als auch die Risiken berücksichtigt.

::: {.teilaufgabe points=15}
**Aufgabe 3.1:** Berechnen Sie den Kapitalwert für jedes der drei Szenarien. Verwenden Sie die Formel: KW = -I₀ + Σ(Ct / (1+i)^t)

*Hinweis: Sie dürfen zur Vereinfachung mit dem Rentenbarwertfaktor arbeiten:*
RBF(10%, 5 Jahre) = 3,791
:::

::: {.teilaufgabe points=12}
**Aufgabe 3.2:** Ermitteln Sie den Erwartungswert des Kapitalwerts unter Berücksichtigung der angegebenen Wahrscheinlichkeiten. Interpretieren Sie das Ergebnis.
:::

::: {.teilaufgabe points=18}
**Aufgabe 3.3:** Das Unternehmen hat die Möglichkeit, zunächst nur 60% der Investition zu tätigen (€1,5 Mio.) und nach 2 Jahren - abhängig von der Marktentwicklung - zu entscheiden, ob die restlichen 40% investiert werden. Dies würde die Flexibilität erhöhen, aber die jährlichen Rückflüsse in den ersten 2 Jahren um 20% reduzieren.

Diskutieren Sie die Vor- und Nachteile dieser Flexibilität und geben Sie eine begründete Empfehlung ab, ob Lisa die gestaffelte Investitionsstrategie verfolgen sollte.
:::
:::

## Integration with Trinity of Depth

Ensure exam covers all three dimensions:

**Theory (Theorie):**
- Conceptual understanding questions
- "Explain the difference between..."
- "Which method is appropriate when..."

**Norms (Normen):**
- Application of standards and rules
- "According to HGB, how should..."
- "Which accounting treatment is required..."

**Practice (Praxis):**
- Calculations and procedures
- "Calculate the..."
- "Prepare the journal entry..."

## Tips for Realistic Case Studies

1. **Use real company names** (anonymized if needed: "TechCorp GmbH" instead of "Company X")
2. **Specific locations** ("Munich" not "a German city")
3. **Realistic numbers** (check if values make sense for company size)
4. **Recent time frame** (2024-2025 for current relevance)
5. **Named protagonists** (real job titles: "CFO Maria Schmidt")
6. **Concrete decisions** (not vague problems)

## Remember

- **Compress Kupp & Mueller methodology** for exam time constraints
- **One minute per point** for time planning
- **Progressive difficulty** within each case
- **Self-contained cases** (all data in text)
- **Closed-ended tasks** (gradable answers)
- **Proper exam-typst formatting** with `.aufgabe` and `.teilaufgabe`
- **Language consistency** throughout
- **Solutions document** separate from exam

Create exams that test application ability while maintaining the pedagogical rigor of good case study design.
